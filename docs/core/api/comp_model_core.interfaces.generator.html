comp_model_core.interfaces.generator

Generator interface.

A generator produces synthetic datasets by simulating a task environment and a computational model. The generator is responsible for enforcing the event/order of operations (e.g., when social observations happen relative to choice and update).

The generator interface is defined at the level of SubjectData and StudyData.
