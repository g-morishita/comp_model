comp_model_core.params.bounds

Box bounds and bounded parameter spaces.

This module provides:

  • Bound: a simple lower/upper bound container.

  • ParameterBoundsSpace: a named vector space with per-parameter bounds.

These are typically used with deterministic optimizers (e.g., SciPy) that support box constraints directly.
